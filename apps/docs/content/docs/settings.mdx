Click the ⚙ tab (or press **⌘,**).

## Display

- **Time format** — 12-hour (3:42 PM) or 24-hour (15:42)
- **Show delta column** — the `+5h` / `-2h` next to each friend

## Working hours

Steppers for start and end of day (0–23). Used by the green/yellow/red coloring in **Find a Time**.

Defaults: **9 → 18** (9am–6pm).

## Refresh

How often the roster re-computes times: **15 / 30 / 60 seconds**. Default 30s. Faster = slightly more battery, but nothing crazy (it’s just recomputing strings).

## Startup

- **Launch DenTime at login** — toggle on to have it show up every time you boot your Mac. Uses Apple’s official `SMAppService`, no login items weirdness.

## About

Version and build number, copyright, and a **Quit DenTime** button (also **⌘Q**).

## Keyboard shortcuts

| Shortcut | Action          |

| ⌘2       | Find a Time tab |
| ⌘,       | Settings        |
