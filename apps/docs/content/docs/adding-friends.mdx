## Groups are whatever you want

DenTime lets you bucket friends into **groups** you define. Common ones:

- **Pack** — your close-knit online crew
- **Work** — teammates, clients
- **Family**
- **School**, **Gaming**, **Open source** — whatever

Friends without a group land under **Friends** at the bottom.

## Add someone

1. Menu bar → paw icon → **+ Add friend**
2. Name
3. Timezone (search by city)
4. Group — pick from existing, make a new one on the fly, or leave ungrouped
5. Color dot — purely cosmetic, helps scan

## Edit, move, delete

Right-click any row in the roster:

- **Edit** — change name, zone, group, color
- **Copy current time** — the clipboard string for that person
- **Move to group** — send them to another bucket
- **Delete** — gone

You can also **drag** within a group to reorder.

## "You" row

The bottom of the roster always shows _your_ current time. That row isn’t editable — it uses your Mac’s system timezone.
