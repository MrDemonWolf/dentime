## MVP1 — now (April 2026)

- ✅ macOS menu bar app
- ✅ Friend roster with groups
- ✅ Find a Time with green/yellow/red tinting
- ✅ Copy-to-clipboard summaries
- ✅ 12h/24h, working hours, refresh rate, launch at login
- ✅ Privacy-first — nothing collected

## MVP2 — next

- **Sync** — opt-in, Apple Sign In, private Cloudflare Worker API backing a small D1 database. Your roster follows you.
- **iPhone app** — Home screen widget + lock screen widget.
- **Meetups** — Doodle-style polls for picking a time that works across the pack.
- **iCal / .ics export** — drop meetings straight into any calendar app.
- **Apple Calendar integration** — optional, read-only, EventKit.
- **Sparkle auto-update** — ships before App Store release cycles.
- **Deep links** — `dentime://friend/alex` style.

## Longer term (maybe)

- Android app
- Web dashboard for managing your pack from any device
- Team / organization mode

No promises on dates. MVP2 targets mid-2026.
