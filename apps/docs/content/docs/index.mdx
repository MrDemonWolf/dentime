DenTime is a tiny, fast, free menu bar app for macOS that shows you the current time for everyone in your pack — friends, coworkers, family. When it’s time to schedule, the **Find a Time** tab tells you what 3pm Chicago looks like in Berlin, Tokyo, and London at a glance.

## What this docs site covers

- **Guide** (you’re here): how to install, add friends, and use the app. No jargon.
- **Developers**: monorepo layout, how to build from source, and how to debug issues.

If you’re here because something isn’t working, jump to [Troubleshooting](/docs/troubleshooting).

## Quick start

1. Install DenTime from the App Store (coming soon).
2. Click the paw icon in your menu bar.
3. Hit **+** to add your first friend.
4. Pick their timezone, done.

That’s it. [More detail →](/docs/getting-started)
